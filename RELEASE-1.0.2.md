# GdOS 1.0.2

- Switches desktop icons to Desktop Icons NG for reliable dragging,
  multi-selection and saved grid positions.
- Fixes Shift+Alt input-source switching at GNOME Shell level.
- Hides the GNOME overview after login and opens the desktop directly.
- Adds detailed crash reports after restarting from the GdOS critical-error
  screen.
- Keeps package-manager authorization for several installs instead of asking
  for the password every time.
- Adds safe application removal to the ArcMenu context menu. Required GdOS
  components remain protected.
- Adds GdOS wallpapers to Appearance and follows the selected accent color.
- Refreshes the Plymouth startup screen and Start button icon.

# GdOS 1.0.1

- Restores Alt+Tab, Super+I, language switching and all GdOS shortcuts
  automatically after login.
- Fixes false GdOS update notifications caused by regular Ubuntu packages.
- Restores movable desktop icons with a compact, aligned grid.
- Reworks the power dialog with a native Ubuntu/Yaru appearance.
- Refreshes the Start button and Quick Install application cards.
- Adds the GdOS critical error screen.

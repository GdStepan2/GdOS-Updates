# GdOS 1.0.3

- Исправлено выделение, перемещение и сохранение позиций значков рабочего стола.
- Добавлено переименование через повторный одиночный клик по имени.
- Исправлены `Shift+Alt`, индикатор раскладки и `Super+Space`.
- Исправлено удаление приложений с отображением результата и ошибок.
- Добавлены Wine, запуск EXE двойным кликом и миниатюры встроенных ICO.
- Обновлены меню Пуск, системные иконки и меню GRUB.

После установки обновления выйдите из учётной записи и войдите снова.

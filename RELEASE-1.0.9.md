# GdOS 1.0.9

Срочное исправление рабочего стола.

- Исправлена ошибка, из-за которой двойное нажатие на значки рабочего стола ничего не открывало.
- Запуск файла или приложения теперь всегда выполняется до визуального изменения курсора.
- Индикатор ожидания больше не может заблокировать открытие.
- После установки расширение рабочего стола DING автоматически перезапускается.

Перезагрузка компьютера не обязательна.

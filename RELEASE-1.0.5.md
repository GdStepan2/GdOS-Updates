# GdOS 1.0.5

- Добавлено автоматическое восстановление значков рабочего стола после аварийной перезагрузки и kernel panic.
- GdOS теперь проверяет реальный процесс Desktop Icons NG, а не только статус расширения.
- При необходимости расширение перезапускается повторно и очищается его устаревшая блокировка.
- Диагностика восстановления сохраняется в `~/.local/state/gdos/desktop-repair.log`.

После установки обновления рекомендуется перезагрузить компьютер.

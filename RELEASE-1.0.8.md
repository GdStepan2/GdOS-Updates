# GdOS 1.0.8

Обновление исправляет стабильность рабочего стола и добавляет инструменты ввода Windows.

## Изменения

- Устранено падение GNOME Shell: расширение уведомлений больше не подменяет внутренние методы анимации и не перехватывает глобальные события клавиатуры.
- Отключены повторяющиеся системные окна Apport для уже обработанных сбоев.
- Для Wi-Fi отключено энергосбережение, вызывающее случайные разрывы соединения.
- Для адаптеров `e1000/e1000e` отключаются проблемные offload-функции, связанные с ошибкой `Tx Unit Hang`.
- Полностью установлена новая синяя тема загрузки Plymouth GdOS вместо экрана Ubuntu.
- При двойном нажатии на значок рабочего стола курсор временно меняется на индикатор загрузки.
- `Win+V` открывает историю буфера обмена с поиском и очисткой.
- `Win+.` открывает панель эмодзи и вставляет выбранный символ в активное приложение.
- `Shift+Alt` теперь переключает источник через GNOME, поэтому индикатор языка соответствует реальной раскладке.

После установки обязательно перезагрузите компьютер.
